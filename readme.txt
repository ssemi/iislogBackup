※ 사용 정보
		로그 파일을 
		일일 / 주간 / 월간 / 강제 (으)로 압축 백업한 뒤에
		압축 백업 파일을 셋팅한 폴더로 옮기고
		압축 파일안의 로그 파일들을 삭제 하고
		완료 되었다는 메일을 보내는 
		스크립트 입니다.

		각각의 프로세스 안에는 리포팅을 할 수 있도록 만들어졌습니다.
		
		테스트 해 본다고 이래저래 시험해본다면 곤란합니다.

※ 사용 방법
	1. 7z.exe 파일을 windows\system32 폴더에 copy 해 넣습니다
	2. 시작 - 실행 - cmd
	3. (cmd 안에서) cscript iisLogBackup.vbs
	4. 윈도우 작업 스케쥴러를 사용해서 백업 처리를 시킨다

	** CScript 가 안되시는 분들은 
	Windows Script Host Version 5.6 을 Download 받으셔서 설치하시면 CScript 의 사용 가능
	** http://www.microsoft.com/downloads/details.aspx?familyid=E74494D3-C4E1-4E18-9C6C-0EA28C9A5D9D&displaylang=ko
